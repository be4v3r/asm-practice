# Практична 4. Сума цифр і робота з діленням (unsigned)

**Мова:** Assembler i386 (NASM)  
**ОС:** Debian Linux  
**Ввід/вивід:** лише `int 0x80` (sys_read/sys_write/sys_exit)

## Мета
Закріпити unsigned-ділення div, роботу із залишком та прості цикли.

## Вхідні дані
Одне додатне число x (1..2_000_000_000).

## Вихідні дані
Два рядки:
1) sumDigits(x)
2) len(x)
Третій рядок (додатково): sumDigits(x) + K

## Що потрібно використати
- div (unsigned): EDX:EAX / r32
- цикл while x > 0
- EDX обнуляти перед div
- підпрограми atoi/itoa (можна повторно використати з попередньої практики)

## Персоналізація (обов’язково)
K = 1 + (Y3 + Y4) mod 9
де Y3/Y4 — 3-я та 4-а цифри року народження.

### Позначення персональних параметрів
- `Y3` — третя цифра року народження (наприклад 2004 → 0)
- `Y4` — четверта цифра року народження (2004 → 4)
- `D` — день народження (1..31)
- `M` — місяць народження (1..12)
- `N` — порядковий номер літери **'n'** у прізвищі (1..len); якщо 'n' немає — `N = 3`
- `K = 1 + (Y3 + Y4) mod 9`
- `S = D + 10*M + 100*Y4`

## Критерії до коду
